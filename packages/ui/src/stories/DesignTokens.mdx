# Design Tokens

Mande Design System uses CSS custom properties for all design decisions.

## Colors

### Neutral
| Token | Usage |
|-------|-------|
| `neutral-white` | Backgrounds, surfaces |
| `neutral-50` – `neutral-900` | Text, borders, fills |
| `neutral-black` | Primary text |

### Semantic
| Palette | Usage |
|---------|-------|
| **Primary** | Brand, CTAs |
| **Teal** | Links, accents |
| **Success** | Positive states |
| **Negative** | Errors, destructive |
| **Warning** | Caution states |
| **Info** | Informational |
| **Blush** | Soft accents |

## Typography

| Scale | Size | Weight |

| H3 | 20px | 600 |

## Spacing

Tailwind v4 default multiplier: `N × 4px`. Numeric scale from `0` (0px) to `20` (80px).

See **Foundations / Spacing** for the full visual reference and Figma mapping.

## Radius

Custom `@theme` tokens from `radius-0` (0px) to `radius-full` (1000px).

See **Foundations / Border Radius** for the full visual reference and Figma mapping.

## Shadows

7 elevation levels from `shadow-2xs` (subtle 1px border-like) to `shadow-2xl` (high-elevation overlay).

See **Foundations / Shadows** for the full visual reference and value breakdown.
